#DeskTop

## 1.关于程序

本程序为windows桌面图标管理程序，现阶段主要功能有：

1. 桌面图标位置备份功能
2. 桌面图标位置还原功能
   - 不久将带来桌面图标版贪吃蛇游戏功能
   - 以及桌面图标实现像素画时钟功能

## 2.开发环境

IDE ：VS 2019

框架：MFC

系统：windows xp以上版本

开发语言：C/C++
